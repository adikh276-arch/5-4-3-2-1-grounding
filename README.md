# 5-4-3-2-1 Grounding

A simple grounding exercise app to help manage anxiety and stress.

## Project info

This project provides a guided 5-4-3-2-1 grounding technique:
1.  **5** things you can see
2.  **4** things you can touch
3.  **3** things you can hear
4.  **2** things you can smell
5.  **1** thing you can taste

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1.  **Clone the repository**:
    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Navigate to the project directory**:
    ```sh
    cd 5-4-3-2-1-grounding
    ```

3.  **Install dependencies**:
    ```sh
    npm install
    ```

4.  **Start the development server**:
    ```sh
    npm run dev
    ```
